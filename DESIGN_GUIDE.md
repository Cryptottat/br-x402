# BRX402 디자인 가이드

## 🎨 컬러 팔레트

### Primary Colors (주요 색상)
- **Primary Orange**: `#FF6B35` (RGB: 255, 107, 53)
  - 메인 브랜드 컬러
  - 호버 효과, 액센트, 강조 텍스트에 사용
  - 로고, 링크, 버튼 호버 상태

- **Secondary Orange**: `#FF8C42` (RGB: 255, 140, 66)
  - 그라데이션 보조 색상
  - Primary Orange와 함께 그라데이션 효과에 사용

### Background Colors (배경 색상)
- **White Background**: `#ffffff` (RGB: 255, 255, 255)
  - 전체 페이지 배경색
  - 카드, 섹션 배경

- **Dark Text**: `#1a1a1a` (RGB: 26, 26, 26)
  - 기본 텍스트 색상
  - 제목, 본문 텍스트

### Accent Colors (강조 색상)
- **Orange with Opacity**:
  - `rgba(255, 107, 53, 0.2)` - 테두리, 경계선
  - `rgba(255, 107, 53, 0.15)` - 미묘한 배경, 그리드 라인
  - `rgba(255, 107, 53, 0.1)` - 매우 미묘한 배경
  - `rgba(255, 107, 53, 0.05)` - 최소 배경
  - `rgba(255, 107, 53, 0.03)` - 거의 투명한 배경

- **Text Opacity Variations**:
  - `rgba(26, 26, 26, 0.7)` - 본문 텍스트
  - `rgba(26, 26, 26, 0.6)` - 보조 텍스트
  - `rgba(26, 26, 26, 0.5)` - 비활성 텍스트

## 🔤 타이포그래피

### 브랜드 폰트
1. **Quart Regular** (`Quart-Regular.otf`)
   - 사용처: "BRX402" 브랜드명 전용
   - 폰트 웨이트: Normal (400)
   - 적용 위치:
     - 헤더 로고 "BRX402"
     - 모든 페이지의 "BRX402" 텍스트
     - 로딩 화면 "BRX402"
     - Docs 페이지의 "BRX402" (주황색 #FF6B35)

2. **NORD Bold** (`NORD-Bold.otf`)
   - 사용처: 메인 슬로건
   - 폰트 웨이트: Bold
   - 적용 위치:
     - "Value transfer should be effortless." (메인 슬로건)

3. **NORD Thin** (`NORD-Thin.otf`)
   - 사용처: 서브 슬로건
   - 폰트 웨이트: Thin (100)
   - 적용 위치:
     - "X402 payments should be as easy as drag and drop." (서브 슬로건)

### 시스템 폰트 (Fallback)
- 기본 폰트 스택: `-apple-system, BlinkMacSystemFont, 'Inter', 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue', sans-serif`
- 코드 블록: `'Courier New', 'Monaco', 'Menlo', monospace`

### 폰트 크기
- 로고: `1.5rem` (24px)
- 메인 슬로건: `clamp(2.5rem, 6vw, 5rem)` (40px ~ 80px 반응형)
- 서브 슬로건: `clamp(1rem, 1.5vw, 1.3rem)` (16px ~ 20.8px 반응형)
- 페이지 제목: `clamp(2.5rem, 5vw, 4rem)` (40px ~ 64px 반응형)
- 섹션 제목: `clamp(2rem, 4vw, 3rem)` (32px ~ 48px 반응형)

### Letter Spacing (자간)
- 로고: `0.2em`
- 네비게이션: `0.05em`
- 일반 텍스트: `0.01em`
- 제목: `-0.02em` (음수 자간)

## 🎭 디자인 컨셉

### 전체 테마
- **컨셉**: "Claude Style" - 화이트 & 오렌지 톤
- **스타일**: 미니멀하면서도 디테일이 살아있는 프리미엄 디자인
- **분위기**: 대형 VC 투자 프로젝트 수준의 고급스러운 느낌

### 레이아웃 특징
- **배경**: 순수 흰색 (#ffffff)
- **텍스트**: 진한 회색 (#1a1a1a)
- **액센트**: 주황색 (#FF6B35)
- **여백**: 넉넉한 여백 (4rem 패딩)
- **그리드**: 기하학적 수직/수평선 (20%, 50%, 80% / 25%, 50%, 75%)

### 애니메이션 & 인터랙션
- **Easing**: `cubic-bezier(0.23, 1, 0.32, 1)` (부드러운 이징)
- **호버 효과**: 
  - 색상 변화 (#1a1a1a → #FF6B35)
  - 언더라인 애니메이션 (width: 0 → 100%)
  - 미세한 상승 효과 (translateY(-1px ~ -2px))
- **로딩**: 순차적 페이드인 애니메이션

### 시각적 요소
- **그라데이션 오브**: 마우스 추적 배경 오브 (주황색 그라데이션)
- **기하학적 선**: 수직선 3개, 수평선 3개 (주황색 투명도)
- **블러 효과**: `backdrop-filter: blur(20px) saturate(180%)`
- **그림자**: 미묘한 그림자 (`box-shadow: 0 2px 20px rgba(0, 0, 0, 0.05)`)

## 🎯 로고 디자인 가이드라인

### 필수 요소
1. **텍스트**: "BRX402" (Quart 폰트 사용)
2. **색상**: 
   - 기본: #1a1a1a 또는 #FF6B35
   - 변형: 단색 또는 그라데이션
3. **스타일**: 미니멀, 기하학적, 프리미엄 느낌

### 권장 사항
- Quart 폰트의 특성을 살린 디자인
- 주황색(#FF6B35) 악센트 활용
- 기하학적 요소 포함 가능
- 단순하면서도 기억에 남는 형태
- 다양한 크기에서도 인식 가능

### 사용 금지
- 과도한 장식
- 복잡한 그래픽 요소
- 브랜드 컬러와 어울리지 않는 색상

## 📊 색상 사용 비율
- **White**: 85% (배경)
- **Dark Gray (#1a1a1a)**: 10% (텍스트)
- **Orange (#FF6B35)**: 5% (액센트, 강조)

## 🔤 폰트 사용 규칙
- **BRX402**: Quart Regular (항상)
- **메인 슬로건**: NORD Bold
- **서브 슬로건**: NORD Thin
- **일반 텍스트**: 시스템 폰트 (Inter, Segoe UI 등)
- **코드**: Courier New, Monaco, Menlo
