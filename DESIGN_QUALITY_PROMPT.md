# 새로운 프로젝트 디자인 요청 프롬프트

## 🎯 기본 요청 템플릿

```
다음 디자인 퀄리티와 수준의 웹사이트를 만들어줘:

## 디자인 퀄리티 기준
- **프리미엄 수준**: 대형 VC 투자 프로젝트 수준의 고급스러운 디자인
- **미니멀하면서 디테일**: 단순하지만 세밀한 디테일이 살아있는 디자인
- **미학적 완성도**: 수학적으로 아름다운 레이아웃과 비율

## 필수 디자인 요소

### 1. 색상 시스템
- 메인 배경: 순수 흰색 (#ffffff)
- 텍스트: 진한 회색 (#1a1a1a)
- 액센트 컬러: [프로젝트에 맞는 컬러 선택]
- 투명도 기반 색상 시스템 (rgba 활용)
- 색상 사용 비율: 배경 85%, 텍스트 10%, 액센트 5%

### 2. 타이포그래피
- 커스텀 폰트 사용 (프로젝트 브랜드에 맞는 폰트)
- 반응형 폰트 크기 (clamp 사용)
- 정교한 자간 조정 (letter-spacing)
- 폰트 계층 구조 명확히 (제목, 본문, 보조 텍스트)

### 3. 레이아웃
- 넉넉한 여백 (4rem 패딩)
- 기하학적 그리드 시스템 (황금비율, 3분할 법칙)
- 수직/수평선을 활용한 수학적 아름다움
- 반응형 디자인 (Desktop 1024px+, Tablet 768-1024px, Mobile 768px-)

### 4. 애니메이션 & 인터랙션
- 부드러운 이징 함수: `cubic-bezier(0.23, 1, 0.32, 1)`
- 순차적 페이드인 애니메이션
- 호버 효과:
  - 색상 전환
  - 언더라인 애니메이션 (width: 0 → 100%)
  - 미세한 상승 효과 (translateY(-1px ~ -2px))
- 마우스 추적 인터랙션 (선택사항)

### 5. 시각적 효과
- 그라데이션 오브/블롭 효과 (배경)
- 기하학적 선 (수직/수평선)
- 백드롭 필터: `backdrop-filter: blur(20px) saturate(180%)`
- 미묘한 그림자: `box-shadow: 0 2px 20px rgba(0, 0, 0, 0.05)`
- 블러 효과와 투명도 조합

### 6. 컴포넌트 스타일
- 버튼: 투명 배경 + 테두리, 호버 시 강조
- 카드: 미묘한 배경색 + 테두리, 호버 시 상승 효과
- 링크: 언더라인 애니메이션
- 헤더: 고정 헤더, 블러 배경

## 기술적 요구사항
- React + Vite 기반
- CSS 모듈 또는 styled-components
- 반응형 디자인 필수
- 부드러운 애니메이션 (60fps 유지)
- 접근성 고려 (alt 텍스트, 키보드 네비게이션)

## 디자인 철학
- "Less is More" - 불필요한 요소 제거
- "Details Matter" - 미세한 디테일로 완성도 높이기
- "Mathematical Beauty" - 기하학적 비율과 정렬
- "Premium Feel" - 고급스러운 느낌 전달

- 브루탈리즘 디자인
- 등등...
```

## 📝 구체적인 요청 예시

### 예시 1: 다크 테마 프로젝트
```
위 디자인 퀄리티 기준을 따르되, 다음으로 변경:
- 배경: 진한 회색 (#0a0a0a)
- 텍스트: 밝은 회색 (#f5f5f5)
- 액센트: 네온 블루 (#00d4ff)
- 기하학적 선: 네온 글로우 효과
- 그라데이션 오브: 블루 톤
```

### 예시 2: 그라데이션 중심 프로젝트
```
위 디자인 퀄리티 기준을 따르되, 다음으로 변경:
- 배경: 미묘한 그라데이션 (위에서 아래로)
- 액센트: 보라색-핑크 그라데이션 (#8B5CF6 → #EC4899)
- 카드: 그라데이션 배경
- 버튼: 그라데이션 배경 + 호버 시 밝아짐
```

### 예시 3: 미니멀 블랙 & 화이트
```
위 디자인 퀄리티 기준을 따르되, 다음으로 변경:
- 배경: 순수 흰색 (#ffffff)
- 텍스트: 순수 검정 (#000000)
- 액센트: 없음 (단색만 사용)
- 기하학적 선: 검정색, 두께 2px
- 모든 색상 투명도 제거 (단색만)
```

## 🎨 핵심 키워드 (AI에게 전달할 때)

다음 키워드들을 포함해서 요청하면 더 정확한 결과를 얻을 수 있습니다:

1. **"프리미엄 디자인"** - 고급스러운 느낌
2. **"미니멀하지만 디테일"** - 단순하지만 세밀함
3. **"기하학적 아름다움"** - 수학적 비율
4. **"부드러운 애니메이션"** - cubic-bezier 이징
5. **"투명도 기반 색상"** - rgba 활용
6. **"반응형 타이포그래피"** - clamp 사용
7. **"미묘한 그림자"** - 섬세한 depth
8. **"백드롭 블러"** - glassmorphism 효과
9. **"순차적 페이드인"** - 스태거 애니메이션
10. **"호버 마이크로 인터랙션"** - 작은 움직임
